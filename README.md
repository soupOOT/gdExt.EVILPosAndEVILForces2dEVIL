# gdExt.3DPosAndForces
Gdevelop's extension for forces and positions.


# what does this extension does?

adds to gdevelop functionalites for forces, put objects in front of another or in front of camera.
translates 3D position for 2D planes (useful for objective icons or compass)
adds distance constraint
and more!
